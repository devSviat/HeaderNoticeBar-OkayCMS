# Модуль Header Notice Bar для OkayCMS

Модуль дозволяє створювати та керувати банерами (оголошеннями), які відображаються в шапці сайту. Банер вставляється безпосередньо після `<header class="header">` на всіх сторінках магазину.

## Можливості

- Створення необмеженої кількості банерів через адмін-панель
- Налаштування фону банера:
  - Однотонний колір
  - Градієнт (з двох кольорів або кастомний CSS градієнт)
- Встановлення дат публікації (публікувати з/до)
- Керування активністю банерів
- Якщо є кілька активних банерів, відображається один випадковий
- Підтримка HTML контенту в банерах
- Мультимовність (підтримка різних мов)

## Вимоги

- OkayCMS v4.5.2

## Встановлення

### Варіант 1: Скачування з GitHub

1. Скачайте останню версію модуля з [останнього релізу на GitHub](https://github.com/devSviat/HeaderNoticeBar-OkayCMS/releases/latest).  
2. Розпакуйте архів на сервері в каталог: `{OkayCMS_root}/Okay/Modules/`

### Варіант 2: Встановлення через Git

1. Виконайте команду в директорії `{OkayCMS_root}/Okay/Modules/Sviat`:
   ```bash
   git clone https://github.com/devSviat/HeaderNoticeBar-OkayCMS.git HeaderNoticeBar
   ```
### Результат встановлення

Шлях до модуля має бути таким:
   ```bash
   {OkayCMS_root}/Okay/Modules/Sviat/HeaderNoticeBar/
   ```

Після встановлення перейдіть в адмін-панель → Модулі → Знайдіть "Header Notice Bar" → Натисніть "Встановити"

## Використання

1. Після встановлення модуля перейдіть в адмін-панель → Модулі → Header Notice Bar
2. Натисніть "Додати оголошення" для створення нового банера
3. Заповніть форму:
   - **Назва** - внутрішня назва банера (для зручності в адмін-панелі)
   - **Контент** - HTML контент, який буде відображатися в банерах
   - **Тип фону** - виберіть "Колір" або "Градієнт"
   - **Колір фону** - якщо обрано тип "Колір", вкажіть колір у форматі HEX (наприклад, #ffffff)
   - **Градієнт** - якщо обрано тип "Градієнт", можна вказати два кольори або кастомний CSS градієнт
   - **Позиція** - порядок сортування банерів
   - **Публікувати з/до** - діапазон дат, коли банер буде активним (необов'язково)
   - **Видимість** - увімкнути/вимкнути відображення банера
4. Натисніть "Зберегти"
5. Банер з'явиться в шапці сайту всередині тега `<header>`, перед основним контентом шапки
6. На мобільних пристроях (екрани до 767px) основний контент автоматично отримає відступ зверху, щоб банер не перекривав контент

## Важливо

Якщо створено кілька активних банерів одночасно, на сторінці буде відображатися один випадковий банер з активних.

**Відступ на мобільних пристроях:** Модуль автоматично додає клас `has-header-notice-bar` до `body`, коли є активний банер, та додає відступ `padding-top: 84px` до `.main` на екранах до 767px. Якщо потрібно змінити розмір відступу, відредагуйте файл `design/css/header_notice_bar.css`.
